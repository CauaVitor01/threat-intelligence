# Threat Intelligence

Este repositório reúne estudos e aplicações práticas de **Cyber Threat Intelligence (CTI)** para apoiar atividades de detecção e resposta a incidentes.

O foco é analisar informações sobre ameaças e transformá-las em **conhecimento acionável para operações de segurança**.

## Conteúdo do Repositório

- Análise de campanhas maliciosas
- Mapeamento de técnicas de ataque
- Estudo de indicadores de comprometimento
- Aplicação de inteligência de ameaças em detecção

## Frameworks Utilizados

- MITRE ATT&CK

## Habilidades Demonstradas

- Cyber Threat Intelligence
- Threat Analysis
- Mapeamento de TTPs
- Apoio à detecção de ameaças

## Estrutura

Os projetos incluem relatórios técnicos, análise de ameaças e mapeamento das técnicas utilizadas pelos atacantes.
